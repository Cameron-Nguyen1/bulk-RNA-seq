ks6.r is a bulk RNA seq analysis script inteded to operate on Kallisto pseudoaligner output.
ks6.r is intended to be submitted to a SLURM job scheduler using a submission script.
Installation_Slurm_Submission.sh is used to submit the script to the job scheduler.
	This script contains arguments that dictate how the analysis behaves.
	See ks6.r --help for more information.
